# What is Mounting?
Mounting is the first phase in a React component's lifecycle when it gets created and inserted into the DOM (Document Object Model). This happens when a component is first rendered.

Mounting involves the following lifecycle methods in class components:

--->constructor()
--->static getDerivedStateFromProps()
--->render()
--->componentDidMount()
